Dataset's Features

Company (str) - Laptop Manufacturer

Product (str) - Brand and Model

TypeName (str) - Laptop type

Inches (int) - Screen Size

ScreenResolution (str) - Screen Resolution

Cpu (str) - Central Processing Unit

Ram (str) - Laptop RAM

Memory (str) - Hard Disk & SSD

GPU (str) - Graphics Processing Units

OpSys (str) - Operating System

Weight (str) - Laptop Weight

Price_euros (int) - Target
